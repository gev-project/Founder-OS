# Documentation Standards

Setiap produk wajib memiliki:

- Problem Statement
- Research Documentation
- Product Requirement Document (PRD)
- Roadmap
- Decision Log
- Result & Learnings

Tidak ada produk tanpa dokumentasi.

---

# Decision Log

Keputusan penting harus dicatat.

Format:

Date:
Product:
Decision:
Reason:
Impact:

Example:

Date:
2026-06-04

Product:
DigiMO

Decision:
Menggunakan Telegram Mini App sebagai platform awal.

Reason:
Mempercepat validasi dan distribusi.

Impact:
Mengurangi waktu pengembangan MVP.
