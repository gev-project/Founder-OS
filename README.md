# About

Gev Project adalah sebuah Product Development Ecosystem yang dibangun untuk mengubah permasalahan operasional nyata menjadi produk digital yang berguna, terdokumentasi, dan berkelanjutan.

Berangkat dari pengalaman di lingkungan Enterprise Telecommunications dan operasional B2B, Gev Project mengadopsi pendekatan yang berfokus pada pemahaman masalah sebelum membangun solusi.

Produk tidak dibangun berdasarkan asumsi.

Produk dibangun berdasarkan observasi, validasi, dokumentasi, dan eksekusi yang terstruktur.

# Guiding Principles

## Problems Before Features

Masalah lebih penting daripada fitur.
